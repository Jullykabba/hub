## Packages
framer-motion | Smooth animations and micro-interactions
date-fns | Date formatting for airdrop start/end dates
lucide-react | Icons

## Notes
- Web3 wallet connection is simulated: The user provides a wallet string which is posted to /api/auth/login
- Admin access requires `isAdmin: true` on the user object
- The app uses CSS custom properties for a Nigerian-inspired cohesive color palette (Green/White/Slate)
- Crypto prices are fetched from /api/crypto/prices to power a real-time marquee ticker
